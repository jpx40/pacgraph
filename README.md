# tauri-leptos-example

- [Tauri][tauri_web]
- [Leptos][leptos_repo]

```sh
# Install Tauri CLI
cargo install tauri-cli

# Build and develop for desktop
cargo tauri dev

# Build and release for desktop
cargo tauri build

```
